## 递归去除对象(根据对象中的属性来去除)
```js
recursivefilter(arr, value) {
      return arr.filter(item => {
      	// itme.type 指定 属性
        if (item.type === value) {
          return false;
        }
        //  上面判断[] 下面判断children[] 
        if (item.childs && item.childs.length > 0) {
          item.childs = this.recursivefilter(item.childs, value);
        }
        return true;
      });
    }
```
