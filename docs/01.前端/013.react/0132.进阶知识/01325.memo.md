#  Memo

+ React组件会在两种情况下发生重新渲染。
  + 第一种，当组件自身的state发生变化时。
  + 第二种，当组件的父组件重新渲染时，第一种情况下得重新渲染无可厚非，state都变了，组件自然应该进行渲染，但是第二种情况似乎并不是总那么的必要。
  + react.Memo()方法里写入组件，父组件更新渲染，子组件的props没改变就不会去渲染
