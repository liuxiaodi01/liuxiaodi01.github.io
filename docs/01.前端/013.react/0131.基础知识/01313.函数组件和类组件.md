#  函数组件和类组件

##  函数组件

```jsx
const App=()=>{
    return <div>函数组件</div>
}
export default App;
```

##  类组件

```jsx
class App extends Component {
  render() {
    return <Fragment>类组件</Fragment>;
  }
}
export default App;
```
