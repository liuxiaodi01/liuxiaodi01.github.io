# Fetch API

Fetch 是浏览器中自带的一种发送请求的方式，它是 Ajax 的升级版，相比起 Ajax 来说它更加方便，代码也更加简洁清晰。
Fetch 需要两个参数：1. 请求地址 2. 请求信息 默认发送的是 get
