插件推荐

发表于2021-07-23|更新于2021-08-11
|阅读量:

Chrome

  1. Adblock Plus : 阻止 YouTube™ 广告、弹出窗口并抵御恶意软件
  2. Chrome清理大师: 一键清理您的浏览器缓存和垃圾，保护您的隐私，并使您的浏览器更快，更高效
  3. Convertio: 在线将文件从一种格式转换为另一种格式. 支持超过 2500 种不同的转换
  4. Custom Chrome - Extension Manager : 管理谷歌扩展插件删除或者开启和搜索等功能
  5. FeHelper(前端助手) : 顾名思义前端助手,一般用来美化json格式数据
  6. Google 翻译: 浏览网页时可轻松查看翻译版本。由Google翻译小组提供。
  7. iGG谷歌访问助手 : 神奇的插件,和谷歌助手一样
  8. infinity 新标签页 : 美化标签页,页面壁纸等等,非常实用。
  9. Shortcuts for Google™ : 提供谷歌快捷方式按钮,非常便捷
  10. Tampermonkey : 油猴,怎么说,非常强大,各种插件,自己体会.
  11. Vue.js devtools : vue开发必备
  12. 万能复制(Enable Copy) : 遇到限制复制粘贴和右键的网页是不是很头痛？安装这个插件后，点下插件按钮就能解决了，任何地方都可以复制内容
  13. 即刻图床 : 完全利用各个互联网大厂的外链图片来做的免费图床，速度快
  14. 哔哩哔哩助手：bilibili.com 综合辅助扩展 : 哔哩哔哩弹幕网辅助扩展，拥有视频区和直播区的人性化功能

vscode

  1. A-super-comprehensive : 懒人必备,代码补全,代码提示,贴心,还有满满几页必备的笔记,可以通过右键查看,最重要的是中文注释!!!!!
  2. Auto Close Tag : 闭合标签
  3. Auto Rename Tag : 标签自动补全,修改标签时,后面的标签会跟着你改的一起修改,非常强大
  4. Bracket Pair Colorizer : 为匹配的括号着色的可定制扩展
  5. Dracula Official : 主题,用习惯感觉非常不错
  6. Fix VSCode Checksums : 调整校验和更改后的核心文件 可以把background安装使用后使用一下这个,非常nice
  7. Fluent Icons : vscode产品图标
  8. Hacker Sounds : 可以听听敲击声音,写代码很有动力
  9. Material Icon Theme : 文件图标
  10. Path Intellisense : 路径提示
  11. px to rem & rpx (cssrem) : 可以转换px成rem
  12. Better Comments : md文件预览,笔记等等功能
  13. 好了,好多网上都必备的我就不介绍了!!!
